Speed

Number of seconds that textition animation should take to complete.

Values positive numbers
Defult 1
Usage $(element).textition({
 speed: 2.3
})

Animation

Sets the pace of action within a transition.

Values ease, ease-in, ease-out, ease-in-out, linear
Default ease
Usage $(element).textition({
 animation: 'ease-in-out'
})

Handler

The event handler which start transition.

Values javascript handlers, false
Default click
Usage $(element).textition({
 handler: 'mouseenter mouseleave'
})

Perspective

CSS3 perspective property for axis Z

Values natural numbers
Defult 300
Usage $(element).textition({
 perspective: 700
})

Map

Transition distance for each axis in pixels

Values natural numbers
Default x: 100, y: 50, z: 1000
Usage $(element).textition({
 map: {
  x: 200,
  y: 0,
  z: 200
 }
})

Autoplay

Automatic text changes

Values true, false
Default false
Usage $(element).textition({
 autoplay: true
})

Interval

Interval between automatic text changes

Values positive numbers
Default 3
Usage $(element).textition({
 autoplay: true,
 interval: 1.5
})

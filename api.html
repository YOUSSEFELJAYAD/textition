textition.js

jQuery plugin for text transitions

About Documentation Examples

Speed

Number of seconds that textition animation should take to complete

Values natural numbers
Defult 1
Usage $(element).textition({
 speed: 2
})

Animation

Describes how the speed during a transition will be calculated

Values ease, ease-in, ease-out, ease-in-out, linear
Default ease-in
Usage $(element).textition({
 speed: 2
})

Handler

Describes how the speed during a transition will be calculated

Values javascript handlers
Default click
Usage $(element).textition({
 speed: 2
})

jQuery plugin for text transitions

Initialization To use textition.js you need jQuery library version 1.6 or later. The initialization code can be inserted into container or . Usage Add method textition() to the object that contains text for transition. $(document).ready(function() { $('#example').textition(); }); The text must be enclosed in tag . Their number depends on the user's needs.
